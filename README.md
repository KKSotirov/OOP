# Обектно-ориентирано програмиране (ООП) 🚀

Това хранилище съдържа задачи, проекти и материали, свързани с подготовката ми по курса по **Обектно-ориентирано програмиране** във ФМИ.

## 📂 Структура на проекта

Репозиторието е организирано спрямо етапите на подготовка и текущите теми:

* **`Exercises/`** – Основна папка за упражнения и подготовка за изпити.
    * `Kontrolno 1/` – Практически задачи върху първата част от материала:
        * Управление на динамична памет (Правило на 3/5 / Rule of 3).
        * Конструктори, деструктори и капсулация.
    * `Kontrolno 2/` – Задачи и подготовки, фокусирани върху:
        * Наследяване и полиморфизъм.
        * Хетерогенни контейнери.
        * Работа с виртуални функции и чист виртуален полиморфизъм.
* **`OOP Project.cpp`** – Самостоятелен практически проект за курса.

## 🛠️ Използвани технологии
* **Език за програмиране:** C++ (стандарт C++11 / C++14 / C++17)
* **Среда за разработка:** Visual Studio Code (VS Code)
* **Версионен контрол:** Git & GitHub

## 📝 Ключови ООП концепции в кода
- [x] Работа с потоци и текстови файлове (`fstream`)
- [x] Коректно управление на ресурси (RAII) и големите 4/5
- [x] Капсулация и модификатори за достъп (`private`, `protected`, `public`)
- [x] Наследяване и композиция
- [x] Шаблони (Templates) и генерично програмиране
